# YourMach Solutions

Sito web aziendale per YourMach Solutions - Consulenza commerciale e soluzioni su misura per aziende.

## 🚀 Deploy su Netlify

1. Vai su [app.netlify.com](https://app.netlify.com)
2. Clicca "Add new site" → "Import an existing project"
3. Seleziona GitHub e questo repository
4. Clicca "Deploy site"

## 📁 Struttura

- `index.html` - Pagina principale
- `*.jpg` - Immagini di sfondo
- `assets/` - CSS e JavaScript

## 🎨 Tecnologie

- React + Vite + Tailwind CSS
- GSAP Animations
